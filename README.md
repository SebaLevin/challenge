## General info

This project is an NodeJs API and was creted as challenge for a job interview.

## Technologies

Project is created with:
  *  express: "^4.18.2",
  *  jsonwebtoken: "^9.0.0",
  *  mysql2: "^3.2.0",
  *  sequelize: "^6.29.3"


## Setup
To run this project, install it locally using npm:

```
$ cd ../challenge
$ npm install
$ npm start
```
